# Changelog

本项目遵循 [Semantic Versioning](https://semver.org/)；当前仍处于 Alpha 阶段。

## [0.1.0-alpha.1] - Unreleased

### Added

- 私有素材库、网页剪藏、语音转写和个人风格档案。
- 选题、带引用草稿、多平台转换和发布记录。
- 选题编辑台支持在 Markdown 模式下使用顶栏格式工具快速加粗、斜体、插入标题、列表、引用、代码块、超链接和表格，并允许直接在 Markdown 光标位置上传插入图片或视频。
- 宿主机持久化、旧卷迁移、完整备份与恢复。
- 首用户注册锁定、凭据加密和安全自托管默认值。

### Fixed

- 修复选题工作台（TopicWorkspace）在编辑器实例未就绪或销毁时，访问 `editor.commands` 导致页面报错崩溃的问题。
- 修复工作台顶部按钮（如“完成写作，进入待发布”）在特定宽度下文案折行、压缩变形的显示缺陷。

### Security

- 内部 AI、数据库和缓存服务不再默认暴露宿主机端口。
- 云模型密钥改为 AES-256-GCM 加密保存。
