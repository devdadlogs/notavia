# Security Policy

## Supported Versions

当前仅维护最新的 `0.1.x Alpha` 版本。Alpha 版本不承诺企业级高可用，公开部署必须使用 HTTPS、强随机密钥和受控注册策略。

## Reporting a Vulnerability

请不要创建公开 Issue。通过 GitHub Security Advisory 的 **Report a vulnerability** 私下报告，并提供受影响版本、复现步骤、影响范围和建议修复。维护者会在 7 天内确认，在修复可用前避免公开利用细节。

自托管实例的数据由实例运营者管理。软件维护者不会因安装 Notavia 自动取得实例中的内容或密钥。
